# Simple Database

# Installation
To clone this repository, execute  
```git clone https://github.com/JanBloch/simple-database.git```  

# Contribution
To contribute to this repository, create a [fork](https://help.github.com/en/enterprise/2.13/user/articles/fork-a-repo#:~:text=A%20fork%20is%20a%20copy,point%20for%20your%20own%20idea.) of the repository. From the fork you can create a [pull request](https://help.github.com/en/github/collaborating-with-issues-and-pull-requests/about-pull-requests). Currently, there are no rules for pull requests, so just try to describe your work.
